# hwameistor-operator
Operator that manages HwameiStor.

* LCM for HwameiStor components
  * LocalStorage
  * LocalDiskManager
  * Scheduler
  * AdmissionController
  * Evictor

* Ensure HwameiStor Ready For Use after installing
  * Auto Claim Disk

* Manage HwameiStor runtime config
  * Admission Config
